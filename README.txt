Soulink • GitHub Pages bundle

Archivos:
- index.html  -> app principal (lee Google Sheets 'Profiles' con gviz)
- 404.html    -> copia de index.html para que /@usuario funcione en GitHub Pages
- .nojekyll   -> desactiva Jekyll para que no procese archivos

Uso rápido:
1) Publica tu hoja en Google Sheets: Archivo -> Publicar en la web -> Hoja: Profiles.
2) Sube estos archivos a un reposorio en GitHub.
3) En Settings -> Pages -> Source: Deploy from a branch -> main / (root).
4) Abre https://<tu-usuario>.github.io/@manu360 (o ?u=manu360).
